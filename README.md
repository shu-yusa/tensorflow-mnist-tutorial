# tensorflow-mnist-tutorial
Tensorflow MNIST tutorial.

## Requirements
* Python 3.6.3
* pyenv
* virtualenv

If not installed, you can install according to the following.
```bash
brew install pyenv
pyenv install 3.6.3
pyenv install virutalenv
```

## Setup
```bash
virtualenv env
source env/bin/activate
pip install -r requirments.txt
```

## Run the calculation
```bash
python cnn_mnist.py
```
